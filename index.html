What problems does responsive design solve?

The main problem responsive design solves on the web is the issue of usability across screens of varying widths. This of course was not necessarily and issue before phones and tablets became a common medium for users to access the internet. Now that they are common place in the average household, websites must adapt(visually) to make sure all of the content is visible and user friendly on each device that a consumer or user may use. For instance, any type that has been made in to a jpg and then set to a width of 100% may shrink to such a small size on a mobile screen that it would be illegible. Therefore, the text should not be made into an image, but instead pure html and set to REM units so it can scale naturally with the users screen size. Advances in responsive techniques are continuously moving forward, however, like most web advancements, the code that you write can only be as up to date as the browsers that the general public are using to interpret your code. Essentially, it is unwise to jump into new coding techniques until they are supported by all modern browsers.
